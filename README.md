#     Chataigne-module-Telegram-Sender
This is a simple Chataigne custom module for sending messages via Telegram Bot API. More info about Chataigne : https://benjamin.kuperberg.fr/chataigne

To create a Telegram Bot you need to message "BotFather" in Telegram.
You can create a new bot with the command '/newbot'. 
The Bot will then ask you for a Name AND Username. You will then receive your new Bot token.

You need to add this bot to a new or existing group.
For the module to work you will need to enter a valid GroupID. You can open the the group chat in https://web.telegram.org and you will see the ID in the URL right after '/a/#'. The GroupID starts with an '-'
